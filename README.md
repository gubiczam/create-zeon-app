# create-zeon-app

Bootstrap a new [Zeon](https://github.com/gubiczam/zeonjs) project with a single command.  
Zero-config starter for building apps with the **Zeon framework**.

## Quickstart

```bash
npx create-zeon-app my-app
cd my-app
pnpm install
pnpm dev
```
